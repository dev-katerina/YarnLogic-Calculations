# YarnLogic-Calculations (Расчетный сервис)

Сервис решает задачу расчетов в вязании и формировании узоров. Является частью приложения для вязания

Планируется реализовать следующие инструменты

1. Инструмент для создания узора крючком
2. Инструмент для создания узора спицами
3. Калькулятор плотности
4. Калькулятор перевода из одной плотности в другую
5. Возможность сохранить узор
6. Сохранение вычислений
